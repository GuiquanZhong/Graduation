# GraduationVue 启动指南

## 项目概述
smart-forum-frontend 是基于 Vue 3 + Vite 构建的智慧论坛前端项目，使用 Element Plus 组件库，通过 Vite 代理连接后端 API（http://localhost:8088）。

## GraduationVue/smart-forum-frontend - smart-forum-frontend

### 快速启动

```bash
cd GraduationVue/smart-forum-frontend
npm run dev
```

**启动后访问**：http://localhost:5173

```yaml
subProjectPath: GraduationVue/smart-forum-frontend
command: npm run dev
cwd: GraduationVue/smart-forum-frontend
port: 5173
previewUrl: http://localhost:5173
description: Vue 3 + Vite 智慧论坛前端，Element Plus UI，后端 API 代理至 http://localhost:8088
```
